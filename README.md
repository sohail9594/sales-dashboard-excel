# 📊 Sales Dashboard Project (Excel)

## Overview
This project involves building an interactive sales dashboard using Excel to analyze business performance.

## Key Features
- Total revenue KPI
- Monthly sales trend analysis
- Sales by region
- Top 5 products analysis
- Interactive slicers (Region, Category)

## Tools Used
- Microsoft Excel (Pivot Tables, Charts, Slicers)

## Insights
- Sales peak during the end of the year
- Certain regions contribute majority of revenue
- Top 5 products generate a large share of total sales
